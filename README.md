# ER Finance

This project was scaffolded manually because `npm` was not available in the environment.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)

## Setup

1.  Open a terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Included Dependencies

- React + Vite
- Tailwind CSS (with custom shadow config)
- Recharts
- Lucide React
- clsx
